Symmetric difference and Summary of Set Operations:

The symmetric difference of two sets is a set operation that results in a new set containing elements that are in either of the original sets but not in their intersection. In other words, it includes elements that are unique to each set. The symmetric difference is denoted by the symbol Î”.
